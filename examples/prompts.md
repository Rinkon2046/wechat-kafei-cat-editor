# 示例 Prompt

## 整理初稿

```text
用 $wechat-kafei-cat-editor 把这篇 Markdown 初稿整理成一篇完成度较高的微信公众号文章，生成 2.35:1 封面、2-3 张正文咖啡猫配图、最终 HTML 和 section-only 复制片段。
```

## 改写来源页

```text
用 $wechat-kafei-cat-editor 改写这个来源页。不要照搬官网结构，压缩宣传模块，把它整理成适合公众号阅读的 3-5 个章节，并生成封面、正文配图和可复制 HTML。
```

## 生成案例文章

```text
用 $wechat-kafei-cat-editor 把这份客户案例整理成公众号文章。保留具体数字、角色和工作流，删掉重复 CTA，生成一张微信封面和三张解释关键流程的咖啡猫配图。
```

## 没有来源时自动省略引用

```text
用 $wechat-kafei-cat-editor 编辑这篇 newsletter 初稿。如果没有识别到来源或引用信息，直接省略引用部分，不要向我索要参考资料。
```
