# Example Prompts

## Polish a Draft

```text
Use $wechat-kafei-cat-editor to turn the attached Markdown draft into a polished WeChat article. Generate a 2.35:1 cover, 2-3 Kafei Cat body illustrations, final HTML, and a section-only copy snippet.
```

## Rewrite a Source Page

```text
用 $wechat-kafei-cat-editor 改写这个来源页。不要照搬官网结构，压缩宣传模块，把它整理成适合公众号阅读的 3-5 个章节，并生成封面、正文配图和可复制 HTML。
```

## Create a Case Article

```text
用 $wechat-kafei-cat-editor 把这份客户案例整理成公众号文章。保留具体数字、角色和工作流，删掉重复 CTA，生成一张微信封面和三张解释关键流程的咖啡猫配图。
```

## Output Without Sources

```text
Use $wechat-kafei-cat-editor to edit this newsletter draft. If no source or citation information is detectable, omit the citation section instead of asking me for references.
```
